The odd distance graph is a case of the unit distance graph, but where there is an edge between two vertices in $\mathbb{R}^2$ if the distance is an odd integer. Now we want to know if the chromatic number of the graph is infinite. \\
Obviously it contains complete subgraphs on three vertices (equilateral triangles), but there does not exist complete subgraphs on four vertices. (Proof?) \\
